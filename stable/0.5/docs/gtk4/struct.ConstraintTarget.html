Struct gtk4::ConstraintTarget

source · 
#[repr(transparent)]
pub struct ConstraintTarget { /* private fields */ }
Expand description

The ConstraintTarget interface is implemented by objects that can be used as source or target in Constraints.

Besides Widget, it is also implemented by ConstraintGuide.
