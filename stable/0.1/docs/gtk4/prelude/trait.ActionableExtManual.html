fn set_action_target(&self, string: &str)

Sets the target of an actionable widget.

This is a convenience function that calls g_variant_new() for format_string and uses the result to call ActionableExt::set_action_target_value().

If you are setting a string-valued target and want to set the action name at the same time, you can use ActionableExt::set_detailed_action_name().
