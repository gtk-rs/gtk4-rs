pub fn accelerator_valid(keyval: Key, modifiers: ModifierType) -> bool
Expand description

Determines whether a given keyval and modifier mask constitute a valid keyboard accelerator.

For example, the GDK_KEY_a keyval plus gdk::ModifierType::CONTROL_MASK mark is valid, and matches the “Ctrl+a” accelerator. But, you can’t, for instance, use the GDK_KEY_Control_L keyval as an accelerator.

keyval

a GDK keyval

modifiers

modifier mask

Returns

true if the accelerator is valid
