fn get_active(&self) -> bool[src]

Queries a ToggleButton and returns its current state. Returns true if the toggle button is pressed in and false if it is raised.

Returns

a gboolean value.

fn set_active(&self, is_active: bool)[src]

Sets the status of the toggle button. Set to true if you want the ToggleButton to be “pressed in”, and false to raise it.

If the status of the button changes, this action causes the ToggleButton::toggled signal to be emitted.

is_active

true or false.

fn set_group<P: IsA<ToggleButton>>(&self, group: Option<&P>)[src]

Adds self_ to the group of group. In a group of multiple toggle buttons, only one button can be active at a time.

Note that the same effect can be achieved via the Actionable api, by using the same action with parameter type and state type ‘s’ for all buttons in the group, and giving each button its own target value.

group

another ToggleButton to form a group with

fn toggled(&self)[src]

Emits the ToggleButton::toggled signal on the ToggleButton. There is no good reason for an application ever to call this function.
