Struct Display

pub struct Display { /* private fields */ }
Expand description

A representation of a workstation.

Their purpose are two-fold:

  • To manage and provide information about input devices (pointers, keyboards, etc)
  • To manage and provide information about output devices (monitors, projectors, etc)

Most of the input device handling has been factored out into separate Seat objects. Every display has a one or more seats, which can be accessed with DisplayExt::default_seat() and DisplayExt::list_seats().

Output devices are represented by Monitor objects, which can be accessed with DisplayExt::monitor_at_surface() and similar APIs.
