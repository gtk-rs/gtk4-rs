List views use widgets to display their contents. You should use Inscription or Label instead Renders text in a cell

A CellRendererText renders a given text in its cell, using the font, color and style information provided by its properties. The text will be ellipsized if it is too long and the GtkCellRendererText:ellipsize property allows it.

If the GtkCellRenderer:mode is CellRendererMode::Editable, the CellRendererText allows to edit its text using an entry.
