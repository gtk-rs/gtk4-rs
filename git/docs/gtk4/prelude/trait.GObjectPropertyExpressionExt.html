Expand description

Trait containing convenience methods in creating [gtk::PropertyExpression] that looks up a property of a glib::Object.

Example

label_expression is a [gtk::Expression] that looks up at Button’s label property.

use gtk::prelude::*;

let button = gtk::Button::new();
button.set_label("Label property");

let label_expression = button.property_expression("label");
