Struct StackSwitcher 

Source 
pub struct StackSwitcher { /* private fields */ }
Expand description

Shows a row of buttons to switch between Stack pages.

It acts as a controller for the associated Stack.

All the content for the buttons comes from the properties of the stacks StackPage objects; the button visibility in a StackSwitcher widget is controlled by the visibility of the child in the Stack.

It is possible to associate multiple StackSwitcher widgets with the same Stack widget.

§CSS nodes

StackSwitcher has a single CSS node named stackswitcher and style class .stack-switcher.

When circumstances require it, StackSwitcher adds the .needs-attention style class to the widgets representing the stack pages.

§Accessibility

StackSwitcher uses the [enum@Gtk.AccessibleRole.tab_list] role and uses the [enum@Gtk.AccessibleRole.tab] role for its buttons.

§Orientable

Since GTK 4.4, StackSwitcher implements Orientable allowing the stack switcher to be made vertical with gtk_orientable_set_orientation().
