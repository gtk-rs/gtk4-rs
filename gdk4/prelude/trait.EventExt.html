pub fn get_history(&self) -> Vec<TimeCoord>[src]

Retrieves the history of the self, as a list of time and coordinates.

The history includes events that are not delivered to the application because they occurred in the same frame as self.

Note that only motion and scroll events record history, and motion events only if one of the mouse buttons is down.

out_n_coords

Return location for the length of the returned array

Returns

an array of time and coordinates

pub fn triggers_context_menu(&self) -> bool[src]

This function returns whether a Event should trigger a context menu, according to platform conventions. The right mouse button always triggers context menus.

This function should always be used instead of simply checking for event->button == GDK_BUTTON_SECONDARY.
